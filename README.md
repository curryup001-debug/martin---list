# Martin Pharmacist Checklist (馬丁清單)

藥師馬丁的藥局工作清單與管理工具。

## 專案功能

- 藥局日常任務清單
- 庫存管理
- 處方記錄
- (目前仍在開發階段)

## 開發指南 (Development)

**前置需求:** Node.js (建議 v18+)

1. 安裝依賴套件:
   ```bash
   npm install
   ```

2. 啟動開發伺服器:
   ```bash
   npm run dev
   ```

3. 建置專案:
   ```bash
   npm run build
   ```

4. 預覽建置結果:
   ```bash
   npm run preview
   ```

## 部署 (Deployment)

本專案使用 GitHub Actions 自動部署至 GitHub Pages。
只要將代碼推送到 `main` 分支，就會觸發自動部署流程。
